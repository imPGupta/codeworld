Introduction to Computer Organization and Design

Computer Organization provides in-depth knowledge of internal working, structuring, and implementation of a computer system.It defines the way the system is structured so that all those catalogued tools can be used properly.

In this section we will study about:
  • The basic structure and operation of digital computer.
  • The design of arithmetic and logic unit and implementation of fixed point and floating-point arithmetic operations.
  • The hierarchical memory system including cache memories and virtual memory.
  • The different ways of communicating with I/O devices and standard I/O interfaces.
